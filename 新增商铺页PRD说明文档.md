# 新增商铺页 PRD 说明文档（详细版）

## 1. 文档信息
- 适用页面：`shop-add.html`
- 页面定位：商铺新建录入页（草稿 / 提交审核）
- 更新时间：2026-04-27

## 2. 页面目标
1. 支持在已有关联企业基础上，完成商铺全量信息录入。
2. 支持“保存草稿”与“提交审核”两类动作，并具备基础校验能力。
3. 支持终端设备弹窗选取、明细填写、自动汇总数量与金额。

## 3. 页面结构
1. 顶部页面说明区（标题+描述）
2. 模块锚点导航（12个模块）
3. 表单主体（企业关联、基础信息、名称、联系、地址、文件、设备、人员等）
4. 底部操作栏（保存草稿 / 提交审核 / 取消）
5. 弹窗区（选择企业、选择服务、选择设备、选择已有人员）

## 4. 全局交互规则
- 锚点导航点击后平滑滚动至对应模块，并随滚动高亮当前模块。
- 页面采用 sticky 顶部与 sticky 锚点导航，确保长表单可用性。
- `SID` 为新增阶段只读展示（提交后生成）。
- 提交审核触发完整校验；保存草稿触发轻量校验。
- 邮箱、URL、金额/数量字段执行格式和范围校验。

## 5. 模块与字段清单

> 字段格式：**字段名｜控件类型｜说明**

### 5.1 关联企业信息
- 选择企业｜弹窗按钮｜通过列表选择已存在企业
- CID｜只读展示｜未选择时显示“未选择”
- 企业名称｜只读展示｜回填企业基础信息
- 企业营业状态｜只读展示
- 企业类型｜只读展示
- 所属地区｜只读展示
- 关联群组｜只读展示
- 重新选择｜按钮｜允许更换关联企业
- 从企业复制信息｜按钮｜将企业地址/联系方式复制到商铺字段

### 5.2 主键 / 关联
- SID｜只读展示｜提交后生成
- CID｜只读输入｜随关联企业自动带入

### 5.3 商铺基础
- 开通Kpay服务｜弹窗多选 + Tag｜支持删除已选服务
- 店面相片 / 存倉相片｜图片上传
- 租金 / 租約年期｜输入
- 私隱政策 / 產品目錄 / 送貨政策｜文件上传
- 门店类型｜下拉
- 聯絡語言｜多选下拉
- 商戶營業狀態(必須)｜下拉
- 特殊業務屬性標籤｜多选下拉
- 退貨政策 / 退貨政策截圖｜输入+上传
- 審核類型 / 上單來源 / 服務類型｜输入/下拉
- 合作夥伴推薦人 / 合作夥伴編號｜输入
- 交單備註｜文本域
- 是否高額商戶｜下拉
- 是否首次申請卡機商戶｜下拉
- 申請過的卡機品牌｜多选下拉
- KPay業務占比｜百分比输入
- 重複簽約原因｜输入

### 5.4 商铺名称
- 当地语言名称 / 简称｜输入
- 英文名称 / 简称｜输入
- 第三语言名称 / 简称｜输入
- 其它名称｜输入
- 所有名称（当地语言 / 英文）｜输入
- 切換商铺顯示名稱（中文 / 英文）｜下拉｜候选项由上方名称字段动态生成

### 5.5 商铺联系
- 固定電話 / 手提電話｜输入
- 商铺電郵地址 / 管理員電郵 / 運營電郵｜邮箱输入
- 商铺接收營銷資訊 / 接收營銷資訊｜下拉
- 商铺微信ID｜输入

### 5.6 商铺地址
- 主要營業地址（当地语言 / 英文）｜文本域
- 營業鋪位｜输入
- 營業國家 / 營業地區｜下拉
- 门店地址（城市 / 州省 / 地址行1-3）｜输入/下拉
- 郵寄地址 / 郵政編碼｜输入

### 5.7 线上资料
- Website(Desktop/Mobile)｜URL输入
- APP(Android/iOS)｜URL输入
- Facebook / Instagram / Openrice｜URL输入
- H5經營網址｜URL输入

### 5.8 商铺文件
- 租約證明 / 商铺LOGO / 營業環境照片三張 等｜文件上传
- 營業環境照片三張｜多文件上传｜最多 3 张
- 商業登記證號碼 / 名稱 / 有效期｜输入/日期
- 证照副本、证明附件等｜文件上传

### 5.9 门店经营特征
- 每宗交易平均金額 / 最大交易額 / 年交易宗數｜数值
- 過往拒付比例 / 退款比例｜百分比
- 拒付/退款平均處理日數｜数值
- 平均每月營業額 / 經營方式｜输入
- 提前收款日數 / 最短送達時間 / 最長送達時間｜输入
- 行業分類 / 業務性質 / 產品服務｜下拉
- 小費功能｜下拉

### 5.10 终端设备
#### 5.10.1 设备明细表
- 新增设备｜按钮｜打开“选择设备”弹窗
- 表格字段：设备型号｜设备分类｜设备序列号｜状态｜数量｜支付金额｜操作
- 数量｜可编辑数值｜按行录入
- 支付金额｜可编辑金额｜按行录入
- 操作｜移除｜删除当前行

#### 5.10.2 汇总与补充字段
- 終端數量｜只读汇总｜自动计算（设备明细数量合计）
- 總金額｜只读汇总｜自动计算（设备明细支付金额合计）
- 終端（自助場景專用）數量｜输入
- 配件數量｜输入
- 每部終端機開戶費｜输入
- 配件申請表｜文件上传
- 自動打印收據(卡類/非卡類)｜下拉

#### 5.10.3 设备弹窗
- 搜索条件：按设备型号、按设备序列号
- 列表字段：设备型号｜设备分类｜设备序列号｜状态｜操作
- 选择后回填明细表，重复选择同一序列号时按钮置灰“已选择”

### 5.11 相关人员
- 新增人员｜按钮｜追加空白行
- 选择已有人员｜弹窗按钮｜从人员池选择后回填
- 列表字段：人员ID｜姓名｜角色身份类型｜是否主联系人｜电话｜邮箱｜操作
- 操作：删除行

### 5.12 其他信息
- 風控類型｜下拉
- 合約期（开始/结束）｜日期范围

## 6. 校验规则
- 提交审核时：
  - 必须已选择关联企业
  - 商戶營業狀態(必須) 不能为空
  - 邮箱字段必须满足邮箱格式
  - URL 字段需以 `http://` 或 `https://` 开头
  - 数值类字段不得为负数
  - 若存在相关人员行，人员ID、姓名、邮箱需合法
- 保存草稿时：
  - 不强制触发完整必填校验，但仍执行基础格式校验

## 7. 底部操作规则
- 保存草稿：提示“草稿已保存（示意）”
- 提交审核：通过校验后提示“提交审核成功（示意）”
- 取消：返回商铺列表页 `shop-list.html`

## 8. 弹窗与联动规则
- 企业弹窗：支持 CID/企业名称/状态/类型筛选与选择回填
- 服务弹窗：多选服务并生成可删除 Tag
- 设备弹窗：支持型号/序列号搜索，选择后进入设备明细
- 人员弹窗：支持 ID/姓名/电话/邮箱筛选，选择后回填人员列表
