# 新增企业页 PRD 说明文档（详细版）

## 1. 文档信息
- 适用页面：`enterprise-add.html`
- 页面定位：企业新建录入页（草稿/提交审核）
- 更新时间：2026-04-26

## 2. 页面目标
1. 支持完整录入企业档案（12大模块）。
2. 支持草稿保存与正式提交审核两种动作。
3. 支持关键字段校验，降低提交失败率。

## 3. 页面结构
1. 顶部信息区（返回、草稿状态、系统生成提示）
2. 模块锚点导航（12个模块）
3. 表单主体（1-12 模块）
4. 底部操作栏（保存草稿/提交审核/取消）
5. 关联群组弹窗

## 4. 全局交互规则
- 锚点导航点击后平滑滚动到对应模块。
- 当前可视模块对应导航按钮高亮。
- `CID / 创建时间 / 更新时间`在新增阶段为只读“系统生成”。
- 提交审核前触发表单校验；保存草稿不强制校验。
- 错误提示统一展示在表单顶部 `error-box`。

## 6. 校验规则
- 必填校验：所有 `required` 字段不能为空。
- 邮箱校验：`email-field` 与 `person-email` 需符合邮箱格式。
- 数字校验：`number-field` 必须可解析为数值。
- 人员校验：相关人员至少1条。

## 7. 底部操作规则
- 保存草稿：直接保存，提示“草稿已保存（示意）”。
- 提交审核：先校验，失败显示错误清单；通过后提示“已提交审核（示意）”。
- 取消：返回企业列表页。

## 8. 弹窗规则（关联群组）
- 支持关键词过滤群组。
- 支持多选。
- 点击“确认”后写回 Tag 区。
- 点击取消/遮罩关闭弹窗（当前实现为取消按钮关闭）。
